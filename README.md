# Projektaufgabe: Data Engineering

## Thema: LLMs for Automating ETL (Extract, Transform, Load) Pipeline Documentation and Code Generation

Beschreibung: In dieser Projektaufgabe sollen die Studierenden eine ETL-Pipeline entwickeln, die große Sprachmodelle (LLMs) nutzt, um die Dokumentation und Code-Generierung für ETL-Prozesse zu automatisieren. Ziel ist es, den Aufwand für die Erstellung und Pflege von ETL-Dokumentationen zu reduzieren und gleichzeitig die Qualität und Konsistenz der generierten Codes zu verbessern.

## Zielsetzung

Ziel der Projektaufgabe ist es, die im Rahmen der Vorlesung "Data Engineering" vermittelten Kenntnisse und Fähigkeiten praktisch anzuwenden. Die Studierenden sollen lernen, wie sie Daten aus verschiedenen Quellen extrahieren, transformieren und laden (ETL-Prozess), um sie für Analyse- und Modellierungszwecke vorzubereiten.

## Rahmenbedingungen:

- Themenspeicher und Reservierung ab Anfang November via BSCW
- Einstiegsliteratur inkl. Quellcode/Workflows im BSCW
- Erläuterung Motivation und theoretische Grundlagen
- Demonstration am Beispiel von Zero Waste-Projektdaten
- Umsetzung mit KNIME und/oder Python
- Präsentation und Diskussion der Ergebnisse
  Bearbeitungszeit mind. 8 Wochen, Abgabe aller Arbeitsergebnisse (Daten, Workflows, Modelle, Programme, etc.) in einem BSCW-Arbeitsbereich Mitte/Ende Februar (zu diskutieren)
